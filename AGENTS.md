## Project Summary
Este projeto é uma landing page de alta conversão para a "Barbearia Modelo", baseada no clone visual da Bonefade Barbers. O template foi localizado para o mercado brasileiro (PT-BR), focado em agendamentos via WhatsApp e uma estética "Modern Vintage Industrial".

## Tech Stack
- HTML5 / CSS3 (Custom Properties)
- Fonts: EB Garamond, Montserrat, Mrs Saint Delafield (Google Fonts)
- JavaScript: Vanilla JS para animações de scroll e scroll-spy
- Deployment: Arquivo único (index.html) para máxima performance e portabilidade

## Architecture
- `index.html`: Arquivo único contendo toda a estrutura HTML, estilos CSS otimizados e lógica JavaScript.
- Ativos: Imagens servidas via CDN/Supabase para carregamento rápido.

## User Preferences
- Localização completa para Português (Brasil).
- Foco em conversão via WhatsApp (Botão flutuante e CTAs).
- Navbar clara (Branco/Creme) com textos escuros para contraste premium.
- Manutenção do efeito Parallax original.

## Project Guidelines
- Manter a agressividade urbana no copywriting ("Cortes de Precisão", "Estilo Único").
- Design responsivo com 'scroll-snap' nativo para carrosséis mobile.
- Zero dependências externas pesadas (sem frameworks JS no arquivo final).

## Common Patterns
- Seções modulares dentro do mesmo arquivo.
- IDs padronizados para navegação suave (#about, #gallery, #services, #footer).
- Botões com cantos retos e transições de cor invertidas.
